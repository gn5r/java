# 推奨コマンド

このプロジェクトでの開発に役立つ主要なコマンドを以下に示します。

*   **プロジェクトのビルド:**
    ```bash
    mvn clean install
    ```
    (ルートの `layered` ディレクトリで実行)

*   **プレゼンテーションモジュールの実行 (Spring Bootアプリケーション):**
    ```bash
    mvn spring-boot:run
    ```
    ( `layered/presentation` ディレクトリで実行)

*   **テストの実行:**
    ```bash
    mvn test
    ```
    (任意のモジュールまたはルートで実行すると、すべてのテストが実行されます)

*   **クリーンアップ:**
    ```bash
    mvn clean
    ```

*   **一般的なシステムユーティリティ:**
    *   `git`: バージョン管理
    *   `ls`: ディレクトリの内容を一覧表示
    *   `cd`: ディレクトリの変更
    *   `grep`: ファイル内のパターン検索
    *   `find`: ファイルの検索
    *   `mvn`: Mavenコマンド
